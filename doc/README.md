# 文档图像倾斜检测与校正系统

## 项目介绍

本项目是基于VC++和OpenCV的文档图像倾斜检测与校正系统，以Hough变换为核心，能自动检测扫描文档的倾斜角度并快速校正，为OCR等后续处理提供高质量图像。同时对比多种算法（投影分析、最小二乘法等）的精度与效率。

## 核心功能

1. **图像预处理**：灰度化、二值化、去噪、边缘提取
2. **倾斜检测**：
   - Hough变换检测直线，计算倾斜角
   - 投影分析法
   - 最小二乘法
3. **倾斜校正**：根据检测角度旋转图像，裁剪黑边并保留完整内容
4. **性能评估**：用不同版面、不同角度的测试集对比检测误差和处理时间
5. **图形用户界面**：支持图像加载、算法选择、结果显示与保存

## 环境配置

1. **OpenCV**：版本4.5.5，安装路径为`D:\opencv`
2. **Visual Studio**：2017或更高版本
3. **平台**：x64

## 编译步骤

1. 打开`SkewCorrect.sln`解决方案文件
2. 在Visual Studio中设置解决方案配置为`Debug`，平台为`x64`
3. 点击"生成"->"生成解决方案"
4. 编译成功后，运行生成的可执行文件

## 使用方法

1. 确保`test.jpg`文件存在于可执行文件所在目录
2. 运行程序后，会显示原始图像和校正后的图像
3. 使用滑块选择不同的倾斜检测算法：
   - 0: Hough变换
   - 1: 投影分析法
   - 2: 最小二乘法
4. 按`s`键保存校正后的图像
5. 按`o`键重新加载图像
6. 按`ESC`键退出程序

## 性能评估

运行`PerformanceEvaluator::evaluate`函数可以评估不同算法的性能，包括角度误差和处理时间。

## 项目结构

- `main.cpp`：主程序，包含图形用户界面
- `ImageProcessor.h/cpp`：图像预处理模块
- `SkewDetection.h/cpp`：倾斜检测模块
- `SkewCorrection.h/cpp`：倾斜校正模块
- `PerformanceEvaluator.h/cpp`：性能评估模块

## 测试建议

1. 使用不同倾斜角度的文档图像进行测试
2. 比较不同算法的检测精度和处理速度
3. 测试不同复杂度的文档图像（如表格、纯文本等）
